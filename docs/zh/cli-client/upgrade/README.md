# iriscli upgrade

## 描述

用来参与软件升级的流程，并可查看当前版本的信息。

## 用法

```shell
iriscli upgrade [command]
```

## 相关命令

| 命令                    | 描述                                               |
| ------------- | ------------------------------------- |
| [submit-switch](submit-switch.md) | 针对软件升级提议发送switch消息|
| [query-switch](query-switch.md)   | 查询switch信息                      |
| [info](info.md)                   | 查询软件升级模块的信息|

## 标志

| 名称, 速记       | 默认值   | 描述            | 必需     |
| --------------- | ------- | ---------------- | -------- |
| --help, -h      |         | 升级命令帮助 |          |
