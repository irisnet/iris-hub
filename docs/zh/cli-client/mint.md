# Mint

Mint模块负责计算通胀的代币数量，并将通胀的代币添加到奖励池中。

## 可用命令

| 名称                              | 描述             |
| --------------------------------- | ---------------- |
| [params](#iris-query-mint-params) | 查询当前mint参数 |

## iris query mint params

该命令用于查询当前mint参数。

```bash
iris query mint params [flags]
```
