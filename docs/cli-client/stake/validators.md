# iriscli stake validators

## Description

Query for all validators

## Usage

```
iriscli stake validators [flags]
```
Print help messages:
```
iriscli stake validators --help
```

## Examples

Query a validator
```
iriscli stake validators
```

After that, you will get all validators' info.
